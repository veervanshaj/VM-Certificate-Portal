📊 Spreadsheet Formatting Guide

The portal accepts standard Excel spreadsheets (.xlsx or .xls). You can upload files in two formats:

Option A: Sheet WITH Headers (Standard)

Create columns using any of these case-insensitive headers:

  • Name: Name, Student Name, name
  • Level: Level, Course, grade
  • Date: Date, Issue Date, date of issue
  • Certificate ID: Certificate ID, ID, Certificate No
  • Email: Email, Email Address

Option B: Sheet WITHOUT Headers (Positional)

If the sheet starts directly with student records on row 1, it maps columns by order:

Col 1
Name
Col 2
Level
Col 3
Date
Col 4
Cert ID
Col 5
Email

💡 Tips on Formats:

  • Dates: Supports standard date strings (e.g. 17-06-2026) and raw Excel serial date numbers (e.g. 46190).
  • Empty Rows: Empty rows or records without student names are ignored automatically.
